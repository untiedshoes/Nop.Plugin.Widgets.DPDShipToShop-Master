Project using the DPD Ship to Shop API to create a plugin for nopCommerce 4.30.

This version has everything contained within the plugin, leaving the nopCommerce source code without any changes.

The customer can choose to have their order delivered to any number of DPD 'shop' locations. The plugin shows the specific store opening times, icons for parking, opening times, disabled parking etc.

Video showing how it works: https://www.youtube.com/watch?v=bCMCrsg9Myg
